## Project Overview

`jpf4826ctl` is a cross-platform command-line tool for controlling JPF4826 fan controllers via Modbus-RTU protocol over RS485. The JPF4826 is an industrial-grade 4-channel PWM DC fan temperature controller with automatic temperature-based speed control and comprehensive fault detection.
